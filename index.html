🌍 What is GeoForge?

GeoForge is a Minecraft Earth server — the map is a 1:1000 scale recreation of the real world. Players build nations, forge alliances, trade resources, and compete for territory on a geopolitically accurate planet. It runs on Java Edition 1.21.11 with the SiegeWar plugin for intense, rewarding geopolitics.

⚙️ How to Join

  • Join the Discord: discord.gg/UwWEeUs8CN
  • Get the server IP from the Discord
  • Launch Minecraft Java Edition 1.21.11
  • Add the server and connect
  • Find a nation or start your own

📋 Useful Commands

  • /n list — list all nations
  • /n spawn — teleport to your nation
  • /pw Ores — teleport to The Ore Shop
  • /t list — list all towns
  • /res info — view your player info
